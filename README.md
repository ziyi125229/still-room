# 静室 · Still Room

可切换场景的**沉浸式自习室**。进入任一场景自动播放对应的自然白噪音，配合极简专注工具，帮你安放此刻的专注。

纯前端、零依赖、单页应用 + 一个图片目录，可离线、可装成 App。

## 功能

- **6 个场景**（按一天光线排列）：晨光森林 · 樱花庭院 · 午后咖啡馆 · 海边日落 · 雨夜书房 · 壁炉雪夜
- **双视觉模式**：每个场景都能在「手绘插画」和「真实照片」之间一键切换
  - 插画版为手绘 SVG（柔和配色 + 蓬松云 + 雾化景深），矢量、离线、清晰
  - 真实版为本地高清照片
- **动效**：Canvas 粒子叠加 —— 雨丝 / 落雪 / 光尘 / 海面波光 / 樱花翻飞 / 篝火火星
- **程序化白噪音**：全部用 Web Audio 实时合成，无音频文件 —— 雨声 / 海浪 / 篝火 / 森林 / 咖啡馆 / 樱花风铃。支持「跟随场景」自动播放，也可关掉后**多音轨自由叠加混音**
- **5 个专注工具**（面板可拖动、记忆位置）：
  - 番茄钟（今日完成计数 · 刷新不中断 · 完成系统通知 · 与待办联动）
  - 正计时（心流）
  - 今日要事（标记“正在做”，番茄记在它头上）
  - 呼吸调息（4-7-8 / 箱式，带读秒）
  - 声音混音
- **极简抗干扰**：鼠标静止自动隐藏界面，全屏沉浸
- **PWA**：可「添加到主屏幕」像 App 打开，离线缓存
- **键盘快捷键**：空格起停番茄 · 数字 1–6 切场景 · Esc 关面板 · F 全屏

## 使用

直接用浏览器打开 `index.html` 即可（点「进入」解锁声音）。
部署时整目录上传即可（含 `img/`、`manifest.webmanifest`、`icon.svg`、`sw.js`）。

## 技术

- 单个 `index.html`（HTML + CSS + JS 内联）+ `img/` 场景照片
- 场景插画：手写内联 SVG
- 声音：Web Audio API 程序化合成
- 离线/安装：Web App Manifest + Service Worker
